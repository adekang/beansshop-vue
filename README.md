# beans shop admin

一个商品管理后台

你可以[点击预览](https://adekang.github.io/beansshop-vue-web/) -- 账号密码【admin】【123456】默认填入输入框的

同时你也会看到我的[源代码地址](https://github.com/adekang/beansshop-vue)

#### 用到的技术栈

- Vue
- Vue-router
- Element-UI
- Axios
- Echarts

获取源码方法

```
// 下载
git clone git@github.com:adekang/beansshop-vue.git

// 下载依赖
yarn install 

// 运行
yarn serve
```
